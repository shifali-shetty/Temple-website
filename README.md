# 🌸 Temple Website

Welcome to **Temple** – a beautifully designed landing page that captures the essence of peace and harmony. This is a simple responsive website featuring an attractive hero section with animated cherry blossom effects.

## 🧘‍♀️ Overview

> "The temple is an attractive tourist place where you can visit and connect to the pure land of the environment. You will experience peace and harmony."

This project focuses on minimal design and smooth front-end effects to create a calming experience.

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Responsive Design**
- **Google Fonts & Remix Icons** – for styling and icons

## 📱 Features

- 🌸 Sakura (cherry blossom) animation
- 🎨 Clean, modern UI
- 📱 Fully responsive on all screen sizes
- 🧭 Navigation bar with menu toggle
- ⚙️ Smooth scroll and layout behavior
